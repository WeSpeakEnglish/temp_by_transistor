# temp_by_transistor

This project will demonstrate the possibility of using an transistor instead of NTC to measure temperature with MCU ADC. 
As for now for beginning:
1. Seeouino XAIO selected
2. We will write an Arduino sketch for easy demonstration
3. BC849C transistor selected, do not ask me why.
4. Will calibrate based on DS18B20+ thermosensor
Updates coming very soon...
